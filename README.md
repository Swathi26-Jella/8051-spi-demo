# 8051 SPI Communication Demo

This project demonstrates basic SPI communication using software-driven signals on the 8051 microcontroller.

## Features

- Byte transmission
- Bit-level communication
- GPIO-based implementation

## Tools Used

- Embedded C
- Keil uVision
- 8051 microcontroller
- Proteus simulation

## Learning Goals

- Serial communication fundamentals
- Timing control
- Embedded driver structure
